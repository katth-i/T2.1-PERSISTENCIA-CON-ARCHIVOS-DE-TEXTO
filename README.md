# T2.1-PERSISTENCIA-CON-ARCHIVOS-DE-TEXTO
El sistema de Gestión de Persistencia de Estudiantes es una herramienta de consola desarrollada en C++ diseñada para administrar registros académicos de forma permanente. La aplicación implementa el uso de flujos de salida mediante la librería &lt;fstream>, permitiendo que la información de los alumnos  no se pierda al cerrar el programa.
